# Introduction

**Use Events** - Events turned into React Hooks.
Read about [Hooks](https://reactjs.org/docs/hooks-intro.html) feature.

## Install

> Note: React 16.7+ is required for Hooks.

### With npm

```sh
npm i use-events
```

### Or with yarn

```sh
yarn add use-events
```

### UMD

```js
<script src="https://unpkg.com/use-events/dist/use-events.min.js"></script>
```

exposed as `UseEvents`

## Quick Examples

```jsx
import { useActive, useFocus } from 'use-events'
import { Button, Input } from './Components'
```

```jsx
const App = () => {
  const [isActive, bind] = useActive();

  console.log(isActive);

  return <Button {...bind}>Click me!</Button>;
};
```

```jsx
const App = () => {
  const [isFocused, bind] = useFocus();

  console.log(isFocused);

  return <Input {...bind} />;
};
```
