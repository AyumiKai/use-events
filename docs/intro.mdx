# Introduction

**Use Events** - Events turned into React Hooks.
Read about [Hooks](https://reactjs.org/docs/hooks-intro.html) feature.

## Install

> Note: React 16.7+ is required for Hooks.

### With npm

```sh
npm i use-events
```

### Or with yarn

```sh
yarn add use-events
```

### UMD

```js
<script src="https://unpkg.com/use-events/dist/use-events.min.js"></script>
```

exposed as `UseEvents`

## Quick Examples

```jsx
import { State, Toggle } from 'react-powerplug'
import { Pagination, Tabs, Checkbox } from './MyDumbComponents'
```

```jsx
<State initial={{ offset: 0, limit: 10, totalCount: 200 }}>
  {({ state, setState }) => (
    <Pagination {...state} onChange={(offset) => setState({ offset })} />
  )}
</State>
```

```jsx
<Toggle initial={true}>
  {({ on, toggle }) => (
    <Checkbox checked={on} onChange={toggle} />
  )}
</Toggle>
```

You can also use a `render` prop instead

```jsx
<Toggle
  initial={false}
  render={({ on, toggle }) => (
    <Checkbox checked={on} onChange={toggle} />
  )}
/>
